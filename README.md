# Km to Mile Converter

A simple **Python 3** GUI application using **Tkinter** to convert kilometers to miles.

## Features

- Convert any number of kilometers to miles.
- Simple and easy-to-use graphical interface.
- Error handling for non-numeric input.

## Requirements

- Python 3.x
- Tkinter (usually included with Python)

## Usage

1. Download the project or copy the code into a file named `km_to_mile.py`.
2. Run the script:

```bash
python km_to_mile.py
